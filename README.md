## Installation

Build slowed-and-reverb-generator from source:

### **1. Clone the repository:**

```sh
git clone https://github.com/raovivek18/slowed-and-reverb-generator
```

### **2. Navigate into the project:**

```sh
cd slowed-and-reverb-generator
```

### **3. Install dependencies:**

```sh
npm install
```

---

## Usage

Run the development server:

```sh
npm start
```

This will start the app locally.
