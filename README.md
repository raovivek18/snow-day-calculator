## Overview

Snow-Day-Calculator is a modern web application that calculates the probability of school closures due to snow. Powered by real-time weather data and advanced algorithms, this tool delivers accurate predictions based on location and school type. Built with **React**, **Vite**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**, it provides a seamless, responsive user experience with beautiful animations and an intuitive interface.

This project requires:

## Installation

Build Snow-Day-Calculator from source:

### **1. Clone the repository:**

```sh
git clone https://github.com/vivek/snow-day-calculator
```

### **2. Navigate into the project:**

```sh
cd snow-day-calculator
```

### **3. Install dependencies:**

```sh
npm install
```

---

## Usage

### **Start the development server:**

```sh
npm run dev
```

This launches the app with hot module reloading at `http://localhost:5173`

### **Build for production:**

```sh
npm run build
```

### **Preview production build:**

```sh
npm run preview
```

### **Lint code:**

```sh
npm run lint
```
